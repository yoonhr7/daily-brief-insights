# Notion 데이터베이스 설정 가이드

## 필수 속성 (Properties)

Daily Brief Insights가 Notion 데이터베이스에 인사이트를 저장하려면 다음 속성들이 필요합니다.

### 기본 속성

| 속성명 | 타입 | 필수 | 설명 |
|--------|------|------|------|
| `제목` | Title | ✅ | 인사이트 제목 (이모지 포함) |
| `도메인` | Select | ✅ | 경제 / IT |
| `상태` | Select | ✅ | 초안 / 발행됨 / 보관됨 |
| `우선순위` | Select | ✅ | 높음 / 중간 / 낮음 |
| `분석일` | Date | ✅ | 분석 수행 날짜 |
| `태그` | Multi-select | ✅ | 관련 키워드 태그 |
| `요약` | Rich Text | ✅ | 인사이트 요약 |
| **`쉬운설명`** | **Rich Text** | ⭐ | **💡 쉽게 이해하기 섹션 (신규)** |

### 도메인별 추가 속성

#### 경제 도메인

| 속성명 | 타입 | 옵션 |
|--------|------|------|
| `이슈유형` | Select | 환율 / 금리 / 증시 / 원자재 / 정책 / 기타 |

#### IT 도메인

| 속성명 | 타입 | 옵션 |
|--------|------|------|
| `변화유형` | Select | 제품출시 / 기술도입 / 기술폐기 / 보안 / 기타 |
| `영향도` | Select | 긴급 / 중요 / 보통 / 낮음 |

---

## 🔧 설정 방법

### 1. Notion 데이터베이스 열기

1. Notion에서 "Daily Brief Insights" 데이터베이스 열기
2. 오른쪽 상단 **"..."** 클릭 → **"Properties"** 선택

### 2. 새 속성 추가: `쉬운설명`

1. **"+ New property"** 클릭
2. 속성명: **`쉬운설명`** (정확히 입력)
3. 타입: **Rich Text** 선택
4. **"Create"** 클릭

### 3. 기존 속성 확인

다음 속성들이 모두 있는지 확인하세요:

#### Select 타입 속성 옵션 설정

**도메인** (Select):
- 경제
- IT

**상태** (Select):
- 초안
- 발행됨
- 보관됨

**우선순위** (Select):

**이슈유형** (Select) - 경제 전용:

- 보안
- 기타

**영향도** (Select) - IT 전용:
- 긴급

## ✅ 설정 확인

설정이 완료되면 다음 명령어로 테스트하세요:

```bash
npm run test:notion
```

성공 메시지가 나오면 설정이 완료된 것입니다!

---

## 📝 참고사항

- `쉬운설명` 속성은 선택사항입니다
- 속성이 없으면 자동으로 건너뛰고 나머지 데이터만 저장됩니다
- `쉬운설명` 속성을 추가하면 더 자세하고 이해하기 쉬운 설명이 저장됩니다
- 각 인사이트에는 이모지(📌)와 함께 구조화된 설명이 포함됩니다

## 🎯 예시 결과

`쉬운설명` 필드에 저장되는 내용 예시:

```
💡 쉽게 이해하기

달러 환율이 오른다는 것은 우리나라 돈의 가치가 떨어진다는 뜻입니다.
수입 제품 가격이 오르고 해외여행 비용이 늘어나지만, 수출 기업들은
해외에서 번 돈을 원화로 바꿀 때 더 많은 돈을 받게 되어 유리합니다.
주로 미국의 금리가 오르면서 투자자들이 달러를 더 선호하게 되었기 때문입니다.
```
